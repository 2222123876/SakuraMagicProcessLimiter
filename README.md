## 🌟 项目介绍
厌倦了进程疯狂占用资源导致电脑卡顿？这款工具专为解决此问题而生！  
以粉紫梦幻的小樱主题打造，界面软萌易操作，支持实时监控进程状态、精准搜索目标进程、自定义资源限制，无论是游戏、设计软件还是后台进程，都能轻松管控，保护系统资源不被过度占用 🧚‍♀️

核心亮点：
- 🎨 专属小樱主题：粉紫配色+可爱图标，告别单调工具界面
- 🚀 低延迟监控：实时刷新进程状态，限制响应迅速
- 🎯 精准管控：支持 CPU 单核心限制+内存上限设置，适配多核心电脑
- 🔍 快速定位：进程名称/PID 双搜索，海量进程中秒找目标

## 📝 使用说明
1. 🚀 启动软件：双击运行（Windows 建议右键「以管理员身份运行」，否则可能无法限制进程）
2. 🔍 选择进程：在列表中直接点击选中，或通过顶部搜索框快速定位
3. ⚙️ 设置限制：
   - 拖动滑块设置「单核心 CPU 使用率」和「内存上限」
   - 点击「⚙️ 设置限制」可精确输入数值
4. ▶️ 开始限制：点击「▶️ 开始限制」，工具自动监控并触发限制逻辑
5. ⏹️ 停止限制：点击「⏹️ 停止限制」，解除对进程的资源管控

## ⚠️ 注意事项
- 🚨 限制功能需要管理员权限！若提示「无权限」，请右键软件选择「以管理员身份运行」
- 🧠 CPU 使用率说明：总使用率（多核心累加，可能超 100%）+ 单核心均值（更直观），限制逻辑基于单核心设置
- 💾 内存限制：Windows 系统支持工作集限制，Linux/macOS 仅支持降低进程优先级
- 🌸 主题适配：所有按钮文字已统一为深紫色，在粉色背景上清晰可见，告别看不清的烦恼
